# 單機版網頁劇本殺 MVP

本專案提供一個直接在瀏覽器打開即可遊玩的簡化劇本殺原型。

## 檔案用途
- **index.html**：單頁應用的入口，包含首頁、角色選擇區與劇本展示區，以及工具列與彈出面板結構。
- **style.css**：頁面樣式，涵蓋英雄區、卡片、角色卡片、劇本區塊與底部工具列。
- **data.js**：集中儲存劇本資料（標題、章節、六個角色的背景、任務、線索與第一章劇本）。
- **script.js**：前端互動邏輯。載入角色列表、處理角色選擇、顯示對應劇本，並控制工具列彈出的資訊面板。

## 使用方式
1. 下載或克隆專案後，直接以瀏覽器開啟 `index.html`。
2. 在首頁按「開始遊戲」，進入角色選擇；點擊一個角色即可看到該角色的第一章劇本。
3. 底部工具列提供「個人宗卷」「當前任務」「已獲得線索」等按鈕，僅展示已選角色的資訊。

無需伺服器、資料庫或外部框架即可運行。若要替換為自訂劇本，可修改 `data.js` 內的角色與劇情文字。
